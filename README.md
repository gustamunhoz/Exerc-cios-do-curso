# Exerc-cios-do-curso
Ao longo deste semestre, adquiri conhecimentos fundamentais em programação, incluindo variáveis, operadores, estruturas condicionais, laços de repetição e funções. Também desenvolvi o raciocínio lógico, a capacidade de resolver problemas e a compreensão da organização e reutilização de código para o desenvolvimento de sistemas.
